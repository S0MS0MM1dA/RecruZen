# RecruZen — Job Recruitment Platform

A simple full-stack job portal with three roles: **Job Seeker**, **Recruiter**, and **Admin**.

## Features
- Job seekers can browse, search, apply, and save jobs  
- Recruiters can post, edit, and manage job listings  
- Admin dashboard for managing users and jobs  
- Secure authentication with password hashing and sessions  

## Tech Stack
- PHP  
- MySQL  
- HTML  
- CSS  
- JavaScript
